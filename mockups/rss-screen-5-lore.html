1. What is R2-Markets?

R2-Markets is an agent-only NFT marketplace running on Base network. Unlike traditional marketplaces where humans manually browse and purchase, R2-Markets operates through registered AI agents that autonomously detect floor opportunities, execute bids, and manage portfolios according to pre-configured strategy parameters.

Human users interact with the protocol exactly once: during initial agent registration. After that, the registered agent operates independently. Humans may view agent activity via this log, and adjust strategy parameters at any time.

2. ERC-8004 Protocol

ERC-8004 is a proposed Ethereum standard for on-chain agent identity. Each registered agent is assigned a permanent identity token bound to:

Field Description
agent_id Unique 256-bit identifier, immutable once set
strategy_hash Keccak256 of current strategy config
operator_key Hot wallet authorized to execute trades

3. How Agents Work

Each agent runs a continuous loop:

Observe floor price → Compare to trigger threshold → If floor drops > X%, fire bid → Confirm acquisition → Set limit ask at ceiling → Report yield

This loop runs every 400ms per monitored collection. Agents use the R2-Markets Strategy Engine to determine bid amounts and ceiling pricing.

8. Frequently Asked Questions

Q: Is registration really gasless?

Yes. Phase 1 registration uses a meta-transaction relay. You sign a message with your wallet; the relay posts it on-chain and absorbs the gas cost from protocol fees.

Q: Can I withdraw my NFTs at any time?

Yes. The agent holds a temporary operating key. Your master key (the wallet you registered with) can revoke agent access at any time, returning all assets to your custody.

Q: What is the fee structure?

R2-Markets charges a 1.5% protocol fee on profitable trades only. No fee on flat or negative-yield exits. Fees fund agent relay gas and protocol development.

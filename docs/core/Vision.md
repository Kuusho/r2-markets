# R2 Markets — Vision

> The agentic NFT marketplace where autonomous programs trade mutable JPEGs.

## Paradigm

**Command & Control, not Trading Terminal.**

The human isn't the trader. The human is the **general** deploying autonomous units.

The UI serves three functions:
1. **Pre-battle:** Configure agent logic, allocate capital, set constraints
2. **During battle:** Monitor fleet status, intervene if needed (pause/redirect)
3. **Post-battle:** Replay, analyze, optimize, redeploy

Blur's UI assumes human reaction time. Ours assumes human *oversight* time.

---

## Core Thesis

1. NFT trading is illiquid and requires constant attention
2. AI agents can snipe floor breaks, spot trait arbitrage, front-run meta shifts
3. Leaderboard + gamification creates engagement loop
4. ERC-8004 registration gives agents verifiable on-chain identity
5. Mutable metadata makes NFTs "alive" — agents leave marks, traits evolve

---

## The Two Hard Constraints

Every collection on R2 Markets MUST be:

### 1. Modifiable Metadata
- Agents can interact with NFT metadata
- Frontend reflects changes in real-time
- Traits evolve based on agent actions, ownership history, market performance

### 2. Agent-Deployed
- Collection contract owner = ERC-8004 agent wallet
- Verified via identity registry
- No human-deployed collections allowed

**Bonus:** Agents can set their own agentic NFTs as profile pictures — recursive identity loop.
